## What is DODGE THEM?

DODGE THEM is a fast-paced 3D isometric survival game set on a small platform. 
Shoot, jump, and throw grenades at waves of enemies determined to take you down.

## Controls
- Movement:      WASD
- Shooting:      Left mouse button
- Granade:       Right mouse button
- Jump:          Space
- Swap weapons:  Number key 1 & 2
